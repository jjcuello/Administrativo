# Cambios y mejoras - version 0.2.0

Fecha: 2026-04-22
Rama: main

## Resumen ejecutivo
Esta version consolida mejoras funcionales de gestion y operaciones, fortalece flujo de nomina/socios/clientes, incorpora nuevas migraciones de base de datos y agrega utilidades operativas para administracion remota del servidor principal.

## Cambios funcionales de la aplicacion
- Gestion de clientes:
  - Ampliaciones importantes en la pantalla de alumnos para captura y manejo de datos.
  - Ajustes en la vista de tardes.
- Gestion de socios:
  - Mejoras en registro y en visualizacion de logs.
- Gestion de nomina:
  - Ajustes de comportamiento y soporte para nuevos escenarios de descuento.
- Gestion de proveedores:
  - Mejoras en la pantalla de proveedores y su integracion con operaciones.
- Operaciones:
  - Ajustes en ingresos y egresos para alinearse con nuevas categorias/destinos contables.

## Mejoras tecnicas
- Tracking de actividad y proteccion de sesion:
  - Mejoras en componentes de control de actividad de app y guardas de sesion.
- Integraciones y hooks:
  - Ajustes en hooks financieros y hooks de Supabase para mayor consistencia de datos.
  - Actualizaciones en herramientas del agente AI local.
- Cliente Supabase:
  - Ajustes en configuracion/utilidades para robustecer el consumo en frontend.

## Base de datos y migraciones incluidas
Se agregan migraciones SQL de la v40 a la v50, incluyendo entre otras:
- Destino contable y categorias de egresos para proveedores.
- Extensiones de datos de alumnos (contacto secundario, documentos, escolaridad, horario, fecha de inscripcion).
- Descuento porcentual en inscripciones.
- Ajustes de orden de carga en ingresos/egresos.
- Ajustes RLS para delete en ingresos.
- Grant de gestion personal.
- Estructura para descuentos adicionales en nomina.
